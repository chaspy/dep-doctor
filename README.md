# dep-doctor
This template is help you to quick implement some CLI using Go.

This repository is contains bellow.

- minimal CLI implementation using [spf13/cobra](https://github.com/spf13/cobra)
- CI/CD
  - golangci-lint
  - go test
  - goreleaser
  - Dependabot
  - CodeQL Analysis (Go)
